## Windows Operating System

### Architecture
- **Kernel Mode**: Hardware abstraction layer, driver management, memory protection
- **User Mode**: Application execution environment, process isolation
- **NTFS File System**: Journaling filesystem with ACL support
- **Power Management**: ACPI integration for sleep/wake cycles and power states

### Core Components
- Windows Subsystem for Linux (WSL)
- .NET runtime environments
- DirectX graphics pipeline
- Windows Terminal and command-line interfaces
