# Minecraft Documentation

## Overview

Minecraft is a sandbox video game developed by Mojang Studios, featuring block-based 3D worlds with survival and creative modes.

### Core Features

- **Survival Mode**: Gather resources, craft items, build structures, fight mobs
- **Creative Mode**: Unlimited resources, flight capability, building without constraints
- **Redstone Engineering**: Complex automation systems and circuit design
- **Multiplayer Support**: Server hosting, modding communities, cross-platform play
- **World Generation**: Procedural terrain with biomes, caves, villages, dungeons
