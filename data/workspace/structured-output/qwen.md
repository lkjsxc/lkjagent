# Qwen Documentation

## Overview

Qwen is a series of large language models developed by Alibaba Cloud, featuring strong reasoning capabilities and multilingual support.

## Architecture

- **Model Family**: Qwen2.5 with 7B to 320B parameters
- **Context Window**: Up to 128K tokens
- **Capabilities**: Code generation, mathematical reasoning, multilingual understanding

### Features

- Native code generation and execution
- Advanced mathematical reasoning
- Multilingual support (100+ languages)
- Long-context processing with precise recall
