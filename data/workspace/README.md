# Workspace Evidence

## Purpose

This is the tracked example root for `/data/workspace`, the directory a running
lkjagent daemon treats as the user workspace.

## Table of Contents

- [AGENTS.md](AGENTS.md): workspace brief loaded into runtime prompt frames.
- [diagnostic-output/](diagnostic-output/README.md): known generated-output
  evidence and failed scaffold records.

## Contract

Content under this root is evidence. It is not a completed owner artifact until
an objective-specific audit records concrete readiness evidence. Scaffold
shape, README links, and catalog entries are structure evidence only.

A generated tree that fails semantic readiness stays under
[diagnostic-output/](diagnostic-output/README.md) with findings instead of being
presented as success.
