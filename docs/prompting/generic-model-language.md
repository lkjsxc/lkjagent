# Generic Model Language

## Purpose

Generic model language owns durable wording for prompts, logs, memory rows,
docs, and generated summaries. lkjagent is provider-neutral by default.

## Preferred Terms

Use these terms unless the owner explicitly asks for a named model topic:

```text
model
local model
model endpoint
reasoning model
assistant model
model profile
model adapter
inference backend
cloud model
local model family
small model
large model
```

## Blocked Durable Claims

Durable output must not claim a named provider, named model, benchmark rank,
capability, or freshness unless the page is an adapter page with a cited source
or a raw evidence fixture.

## Raw Evidence

Raw owner text and raw logs remain available as evidence. Display logs,
prompts, docs, memory rows, and generated summaries use sanitized wording and a
replacement report.
