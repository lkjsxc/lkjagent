# Bias Detection

## Purpose

Bias detection owns skew checks across documentation, artifacts, prompts, and
runtime maintenance. It finds overbuilt areas and underlinked requested topics.

## Biases

- explanatory-overview-over-implementation-contract.
- model-provider-specificity.
- generic-directory-over-semantic-seed.
- reference-pages-without-implementation-links.
- recovery-loop-repeat.
- maintenance-loop-repeat.
- artifact-profile-drift.

## Action Rule

A detected bias becomes either a focused refactor plan, a regression fixture, or
a suppression record. It does not become an immediate broad rewrite.
