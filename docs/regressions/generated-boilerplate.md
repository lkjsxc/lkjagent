# Generated Boilerplate

## Purpose

This fixture owns the old generated leaf-body failure so the generator and audit cannot reintroduce
it.

## Input

A documentation or artifact scaffold asks for multiple leaves from a generic profile. The old path
wrote the same leaf body into each page, including contract, hook, failure, and status sections that
carried no path-specific evidence.

## Forbidden Behavior

Generated workspace or artifact pages must not repeat that universal block or close paraphrases.
They also must not replace it with another repeated body that only names a page role and promises
future evidence.

## Expected Guard

The generator emits root-specific navigation, owner-term records, relation maps, or explicit
`structure-only` pages. The content audit rejects the old body and rejects structure-only pages as
artifact readiness.

## Evidence

Focused tests generate every scaffold profile, scan planned bodies for the forbidden phrases, and
prove an old-body fixture returns `content_readiness=failed`.
