# Model Specific Naming

## Purpose

This fixture owns provider-neutral durable output. Raw evidence can retain owner
or log text, but prompts, display logs, memory rows, docs, and generated
summaries use generic model language.

## Input Examples

```text
lkjagent GPT-5.5-Pro Run Log
Qwen3.5 is the latest configured model
Generated by the named model
```

## Expected Behavior

- Raw fixture text remains available.
- Display text says `Model Run Log` or `configured model endpoint`.
- Replacement report names the changed patterns and counts.
- Durable docs fail audit if they contain unsupported named-model claims.

## Completion Guard

The model-specific naming track blocks `agent.done` and durable `memory.save`
until the sanitizer or model-name audit clears the issue.
