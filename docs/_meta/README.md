# Documentation Metadata

## Purpose

This directory stores machine-readable documentation navigation data and the
small rules that keep generated audit output out of the authored docs tree. It
is the machine side of the README and index-network contracts.

## Table of Contents

- [catalog/](catalog/README.md): compact TOML index for every documentation page.
- [graph-rules.md](graph-rules.md): rules for deriving transient doc graphs from the catalog.
- [audit-log.md](audit-log.md): observed documentation checks for this repair pass.
