# Story Profile

## Purpose

Give an example semantic tree for a story artifact. This profile is an
example, not the only valid story output.

## Example

```text
stories/<semantic-title>/
  README.md
  manifest.md
  premise.md
  cast.md
  world.md
  outline.md
  chapters/
    README.md
    waking-pod.md
    cylinder-pulse.md
    laboratory-of-light.md
    memory-market.md
    first-theft.md
    mirror-in-the-vial.md
    final-choice.md
    world-after-memory.md
  audit.md
```

## Audit

A story audit requires premise, cast, world or setting, outline, chapter
content, README links, and a manifest. Scaffold-only chapter labels are not
content evidence.

## Status

partially implemented through story scaffold and content audit checks.
