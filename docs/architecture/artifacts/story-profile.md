# Story Profile

## Purpose

Define the creative writing scaffold profile for story, novel, manuscript,
character, setting, outline, and scene artifacts.

## Example

```text

  README.md
  catalog.toml
```

## Selection

The creative profile is selected when `kind` or title names story, novel,
manuscript, fiction, character-profile, setting, outline, scene, SF, sci-fi, or
science fiction, unless the request is clearly for technical documentation.

## Audit

A story audit requires premise, cast, world or setting, outline, draft content,
README links, and catalog metadata. Scaffold-only creative labels are not
content evidence.
