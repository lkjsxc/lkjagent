# lkjagent-benchmark Tests

## Purpose

This directory holds focused tests for corpus validation, judge behavior,
metrics extraction, endpoint configuration checks, and report rendering.

## Table of Contents

- [corpus.rs](corpus.rs): task validation and fixture judge tests.
- [metrics.rs](metrics.rs): transcript metric extraction tests.
- [report.rs](report.rs): TSV, markdown, and compare rendering tests.
- [runner.rs](runner.rs): endpoint configuration tests.
