# lkjagent-protocol Source

## Purpose

This directory holds the pure action parser, renderer, and registry table.

## Table of Contents

- [error.rs](error.rs): parser result alias.
- [lib.rs](lib.rs): library root.
- [model.rs](model.rs): action and fault data.
- [parse.rs](parse.rs): strict completion parser.
- [registry.rs](registry.rs): tool parameter table.
- [registry_spec.rs](registry_spec.rs): canonical tool list.
- [render.rs](render.rs): action and context frame renderers.
