# lkjagent-protocol Source

## Purpose

This directory holds the pure action parser, renderer, and registry table.

## Table of Contents

- [error.rs](error.rs): parser result alias.
- [lib.rs](lib.rs): library root.
- [model.rs](model.rs): action and fault data.
- [parse.rs](parse.rs): strict completion parser.
- [registry.rs](registry.rs): tool parameter table.
- [registry_spec.rs](registry_spec.rs): canonical tool list.
- [render.rs](render.rs): action and context frame renderers.
- [tag_line.rs](tag_line.rs): structural tag-line classifier.
- [xml_parse.rs](xml_parse.rs): paired-tag action parser.
- [xml_parse/](xml_parse): paired-tag parser helpers.
- [registry_graph.rs](registry_graph.rs): registry graph source module.
- [registry_personal.rs](registry_personal.rs): registry personal source module.
- [registry_render.rs](registry_render.rs): registry render source module.
- [xml_parse/](xml_parse/README.md): xml parse helper modules.
