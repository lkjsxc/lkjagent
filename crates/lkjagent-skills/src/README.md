# lkjagent-skills Source

## Purpose

This directory holds skill validation, indexing, and loading code.

## Table of Contents

- [index.rs](index.rs): deterministic skill index rendering and degradation.
- [lib.rs](lib.rs): library root.
- [load.rs](load.rs): skill file loading into immutable frames.
- [model.rs](model.rs): skill data and validation report types.
- [validate.rs](validate.rs): shared skill format validator.
- [validate/](validate/README.md): validator helper modules.
