# Personal Store Helpers

## Purpose

Typed SQLite helpers for diary, schedule, and TODO records.

## Table of Contents

- [model.rs](model.rs): input, row, and list-filter structs.
- [validate.rs](validate.rs): pure validation for kinds, statuses, dates, and recurrence.
- [write.rs](write.rs): create, status update, link, event, and FTS writes.
- [read.rs](read.rs): get, list, and FTS search helpers.
