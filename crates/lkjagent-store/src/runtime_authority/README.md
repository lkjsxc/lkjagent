# Store Runtime Authority

## Purpose

This directory owns helper data and codecs for normalized runtime authority
store APIs.

## Table of Contents

- [codec.rs](codec.rs): lossless list encoding for store fields.
- [model.rs](model.rs): input and row data for authority history tables.
- [dense_model.rs](dense_model.rs): dense model source module.
- [dense_read.rs](dense_read.rs): dense read source module.
- [dense_write.rs](dense_write.rs): dense write source module.
- [detail_model.rs](detail_model.rs): detail model source module.
- [detail_read.rs](detail_read.rs): detail read source module.
- [detail_write.rs](detail_write.rs): detail write source module.
- [frame_write.rs](frame_write.rs): frame write source module.
- [read.rs](read.rs): read source module.
- [write.rs](write.rs): write source module.
